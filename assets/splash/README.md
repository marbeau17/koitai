# Splash Screen

Place the following splash file in this directory:

## splash_logo.png
- Size: 512x512 pixels (recommended)
- Format: PNG with transparency
- Design: App logo (heart + moon + stars theme) in white or light color
- This will be displayed centered on the splash screen
- Background color: #0F0A1A (dark background, configured in pubspec.yaml)

## Generation
After placing the splash logo file, run:
```bash
dart run flutter_native_splash:create
```
