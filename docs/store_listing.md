## 詳細な説明 / Full Description

恋のタイミング、ここにある。

コイタイは、数秘術・月の満ち欠け・バイオリズムの3つの要素を掛け合わせて、あなただけの「恋愛ベストタイミング」を毎日お届けする、新感覚の恋愛占いアプリです。

「今日は気になるあの人に話しかけるべき？」
「告白するなら、いつがベスト？」
「デートの予定、この日で大丈夫？」

そんな恋の悩みに、コイタイが答えます。

----------------------------------------------

【毎日の恋愛運スコア】

あなたの生年月日から導き出す数秘術、月の満ち欠けが持つエネルギー、そして身体・感情・知性の3つのバイオリズム。これらを独自のアルゴリズムで組み合わせ、0〜100の恋愛運スコアを毎日算出します。

スコアが高い日は、恋愛のチャンス到来のサイン。大切なアクションは、スコアが高い日に合わせてみましょう。

----------------------------------------------

【カレンダーで恋愛運を一覧チェック】

月間カレンダーに恋愛運の評価が表示されるので、先の予定を立てるのに最適です。

- 恋愛運の高い日がひと目でわかる
- デートや告白のベストタイミングを事前にチェック
- 日付をタップすれば、その日の詳しい運勢を確認

----------------------------------------------

【ペア相性診断】

気になるあの人との相性をチェック。お互いの生年月日を入力するだけで、数秘術に基づく相性スコアと詳しい診断結果がわかります。

- 2人の数秘術的な相性を分析
- 今日の2人の恋愛タイミングの一致度
- 関係性を深めるためのアドバイス

----------------------------------------------

【プッシュ通知で毎日お届け】

毎朝、今日の恋愛運スコアをプッシュ通知でお届け。アプリを開かなくても、今日のコンディションがすぐにわかります。恋のチャンスを逃しません。

----------------------------------------------

無料でも基本的な恋愛運スコアをご利用いただけますが、プレミアムプランではすべての機能を無制限にお楽しみいただけます。
