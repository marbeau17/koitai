# プライバシーポリシー

最終更新日：2026年2月23日

## はじめに

本プライバシーポリシー（以下「本ポリシー」）は、コイタイ（KoiTai）アプリケーション（以下「本アプリ」）における個人情報の取り扱いについて定めるものです。本アプリをご利用いただくにあたり、本ポリシーの内容をご確認ください。

本アプリの運営者（以下「当社」）は、個人情報の保護に関する法律（個人情報保護法 / APPI）および EU 一般データ保護規則（GDPR）を遵守し、ユーザーの皆様の個人情報を適切に管理いたします。

## 収集する情報

本アプリでは、以下の情報を収集します。

### 1. ユーザーが提供する情報

| 情報 | 目的 |
|------|------|
| 生年月日 | 数秘術・バイオリズム・月相に基づく恋愛運の計算 |
| 性別 | 占い結果のパーソナライズ |
| メールアドレス | アカウント管理（Google サインイン / Apple サインインを通じて取得） |

### 2. 自動的に収集される情報

| 情報 | 目的 |
|------|------|
| FCM トークン（Firebase Cloud Messaging） | 毎日の恋愛運プッシュ通知の配信 |
| アプリ利用状況データ | サービス改善のための分析（Firebase Analytics） |
| クラッシュレポート | アプリの安定性向上（Firebase Crashlytics） |
| デバイス情報（OS バージョン、デバイスモデル） | 技術的なサポートおよびサービス最適化 |

## 利用目的

収集した情報は、以下の目的のために利用します。

1. **占い機能の提供** - 生年月日・性別に基づく恋愛運スコアの算出
2. **プッシュ通知** - 毎日の恋愛運をお届けするための通知配信
3. **アカウント管理** - ユーザー認証およびデータの同期
4. **サービス改善** - アプリの品質向上およびユーザー体験の改善
5. **カスタマーサポート** - お問い合わせへの対応

## 第三者サービス

本アプリでは、以下の第三者サービスを利用しています。各サービスは、それぞれのプライバシーポリシーに従って情報を取り扱います。

| サービス | 提供元 | 用途 | プライバシーポリシー |
|----------|--------|------|----------------------|
| Firebase Authentication | Google LLC | ユーザー認証 | [Firebase プライバシー](https://firebase.google.com/support/privacy) |
| Cloud Firestore | Google LLC | データの保存・同期 | 同上 |
| Firebase Analytics | Google LLC | 利用状況の分析 | 同上 |
| Firebase Crashlytics | Google LLC | クラッシュレポートの収集 | 同上 |
| Firebase Cloud Messaging | Google LLC | プッシュ通知の配信 | 同上 |
| Google Sign-In | Google LLC | Google アカウントによる認証 | [Google プライバシーポリシー](https://policies.google.com/privacy) |
| Sign in with Apple | Apple Inc. | Apple アカウントによる認証 | [Apple プライバシーポリシー](https://www.apple.com/legal/privacy/) |
| RevenueCat | RevenueCat, Inc. | サブスクリプション管理 | [RevenueCat プライバシーポリシー](https://www.revenuecat.com/privacy/) |

## データの保存

- **ローカルストレージ**: 占い結果やユーザー設定は、Hive を使用してデバイス内にローカル保存されます。
- **クラウドストレージ**: アカウント情報および占いデータは、アカウントが有効な間、Firebase（Cloud Firestore）に同期・保存されます。
- **保存期間**: アカウントが有効な間、データは保持されます。アカウント削除後、関連するすべてのデータは速やかに削除されます。

## データの安全管理

当社は、個人情報の漏洩、紛失、毀損等を防止するため、以下のセキュリティ対策を講じています。

- Firebase のセキュリティルールによるアクセス制御
- 通信の暗号化（TLS/SSL）
- アクセス権限の最小化

## ユーザーの権利

ユーザーは、以下の権利を有します。

### アカウントおよびデータの削除

本アプリ内の設定画面から、アカウントおよび関連するすべてのデータを削除することができます。削除リクエストを行うと、以下のデータがすべて削除されます。

- Firebase Authentication のアカウント情報
- Cloud Firestore に保存されたユーザーデータ
- デバイス上のローカルデータ

### データへのアクセス・訂正

ご自身の個人情報へのアクセス、訂正、または利用停止をご希望の場合は、下記のお問い合わせ先までご連絡ください。

### GDPR に基づく権利（EU 居住者）

EU に居住するユーザーは、GDPR に基づき、以下の権利を有します。

- **アクセス権**: ご自身の個人データへのアクセスを請求する権利
- **訂正権**: 不正確な個人データの訂正を請求する権利
- **削除権（忘れられる権利）**: 個人データの削除を請求する権利
- **処理制限権**: 個人データの処理の制限を請求する権利
- **データポータビリティ権**: 個人データを構造化された形式で受け取る権利
- **異議申立権**: 個人データの処理に異議を申し立てる権利

### 個人情報保護法（APPI）に基づく権利（日本居住者）

日本に居住するユーザーは、個人情報保護法に基づき、保有個人データの開示、訂正、追加、削除、利用停止、消去および第三者への提供の停止を請求する権利を有します。

## 子どものプライバシー

本アプリは、13歳未満のお子様を対象としておりません。13歳未満のお子様から故意に個人情報を収集することはありません。万が一、13歳未満のお子様の個人情報を収集していることが判明した場合は、速やかに当該情報を削除いたします。

## 本ポリシーの変更

当社は、必要に応じて本ポリシーを変更することがあります。重要な変更がある場合は、本アプリ内で通知いたします。変更後のポリシーは、本アプリ内に掲載した時点で効力を生じるものとします。
